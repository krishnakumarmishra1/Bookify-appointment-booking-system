# 🚀 Bookify - Appointment Booking System

## 📌 Overview
Bookify is a full-stack Django-based appointment booking system that supports multiple service categories like Doctor, Salon, and Consultancy.

Users can browse services, book time slots, and manage appointments with a smooth and modern UI.

---

## ✨ Features

- 🔹 Multi-service system (Doctor, Salon, Consultancy)
- 🔹 Service filtering & search
- 🔹 Time slot booking system
- 🔹 No double booking logic
- 🔹 Past time slots disabled
- 🔹 Payment system:
  - Cash (Cancelable)
  - Online (Non-cancelable)
- 🔹 My Bookings page with status
- 🔹 Admin Dashboard (Bookings + Revenue)
- 🔹 Clean UI with Bootstrap & animations

---

## 🛠️ Tech Stack

- Backend: Django (Python)
- Frontend: HTML, CSS, Bootstrap
- Database: SQLite
- Version Control: Git & GitHub
